🚀 Java Workspace

This repository contains a collection of projects and practice code covering Core Java, Advanced Java, and modern frameworks like Spring and Spring Boot. It serves as a complete workspace for learning, building, and experimenting with Java-based technologies.

📚 Topics Covered
Core Java (OOPs, Collections, Exception Handling, etc.)
Advanced Java (JDBC, Servlets, JSP)
Spring Core
Spring MVC
Spring Data JPA
Hibernate ORM
Spring Boot
Data Structures & Algorithms (DSA)


📂 Project Structure
workspace/
│── Collection-framework-workspace   # Java Collection Framework examples
│── advance-workspace                # Advanced Java concepts
│── dsa-workspace                    # Data Structures & Algorithms
│── java-workspace                   # Core Java programs
│── spring-workspace                 # Spring Core & Hibernate CRUD
│── spring-mvc-workspace             # Spring MVC + JPA Mapping
│── spring-boot-workspace            # Spring Boot applications
│── testworkspaces                   # Practice/test code


🛠️ Technologies Used
Java (JDK 8+)
Spring Framework
Spring Boot
Hibernate
JPA (Java Persistence API)
Maven / Gradle
MySQL (for database projects)


🎯 Purpose
Practice and strengthen Java concepts
Build real-world backend applications
Prepare for interviews (Java + Spring stack)
Understand framework integration (Spring + Hibernate + JPA)
